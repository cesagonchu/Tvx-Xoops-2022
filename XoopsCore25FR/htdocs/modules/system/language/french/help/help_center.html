Bienvenue au : Centre d'aide de XOOPS

Comment utiliser XOOPS ?

XOOPS est une plateforme d'applications Internet écrite en PHP pour la base de données MySQL. Son orientation objet en fait un outil idéal pour développer des sites communautaires de petite ou grande taille, des portails intra-entreprises et corporatifs, des blogs et bien plus encore. Afin d'utiliser XOOPS efficacement, vous devez installer des modules qui fournissent la fonctionnalité que vous recherchez. Il existe de nombreux modules différents dans le dépôt de modules, allant des calendriers aux galeries de photos, en passant par les nouvelles, les tableaux d'affichage et les mises à jour météorologiques.

Comment utiliser le centre d'aide ?

Le centre d'aide a pour but de vous fournir des informations de base sur les différentes fonctionnalités de l'administration. Si vous souhaitez en savoir plus sur la page sur laquelle vous vous trouvez, cliquez simplement sur l'icône d'aide dans le coin droit.

Comment puis-je créer du contenu dans XOOPS ?

Vous aurez besoin d'un module pour cela. Il peut s'agir d'un module d'actualités ou d'un module de contenu.

Comment puis-je obtenir plus d'aide ?
