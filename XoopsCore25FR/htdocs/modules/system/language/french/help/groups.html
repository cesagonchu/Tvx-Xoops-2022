Aide : Groupes

Description

XOOPS intègre un système d'inscription des membres - vous pouvez optionnellement demander aux gens de s'inscrire sur votre site afin d'avoir accès à certaines zones ou fonctions. Par exemple, la plupart des sites exigent que les gens s'inscrivent avant d'être autorisés à contribuer aux nouvelles ou à poster des messages dans les forums.
Les droits d'accès et d'administration d'un site XOOPS sont définis par un système de permission flexible basé sur des "groupes d'utilisateurs". Les groupes par défaut comprennent les utilisateurs anonymes, les membres enregistrés et les webmasters, mais vous pouvez bien sûr définir autant de groupes que vous le souhaitez pour gérer votre site. Chaque groupe peut se voir attribuer un ensemble unique de droits d'accès régissant :

- Quels blocs peuvent être vus
- Quels modules sont accessibles
- Quels modules peuvent être administrés
- Quels aspects de l'administration du système peuvent être modifiés

Vous pouvez modifier le nom de ces groupes par défaut, mais vous ne pouvez pas les supprimer. D'autres groupes d'utilisateurs peuvent également être définis, comme nous le verrons plus tard.

Des groupes personnalisés peuvent également être créés via la page d'administration des groupes.
