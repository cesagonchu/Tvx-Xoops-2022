Aide : Courriels aux utilisateurs

Description

XOOPS possède un système de messagerie intégré qui vous permet d'envoyer un message à l'ensemble de votre base d'utilisateurs enregistrés ou à des sous-groupes de ceux-ci, ou en utilisant des filtres de date. Cela peut se faire sous la forme d'un message privé (c'est-à-dire via le système de messagerie intégré de XOOPS) ou d'un courriel. Une gamme d'options est disponible pour filtrer les sous-groupes d'utilisateurs de votre adhésion.

Vous pouvez également utiliser des balises prédéfinies qui vous permettront de personnaliser votre message.
