Aide : Émoticônes

Description

Les émoticônes sont de petits symboles graphiques que les gens peuvent utiliser pour s'exprimer. Ils peuvent être insérés n'importe où dans le texte d'un article, d'un commentaire ou d'un message, tant du côté utilisateur que du côté administrateur du système. XOOPS est livré avec une collection de 17 émoticônes intégrées. Certaines d'entre elles sont affichées sur les formulaires de soumission dans divers modules (y compris les forums, les nouvelles, les téléchargements, les liens et autres). En cliquant sur une émoticône, celle-ci sera insérée dans le message. Les émoticônes peuvent également être insérées en tapant leurs "codes" uniques (raccourcis clavier), par exemple en tapant ':' '-' 'D' dans un message pour insérer cette émoticône :

La liste complète des émoticônes disponibles sur votre système et leurs codes sont accessibles par l'intermédiaire des émoticônes, où vous vous trouvez actuellement. Vous pouvez y ajouter, modifier et supprimer des émoticônes, et contrôler celles qui sont affichées dans les formulaires de soumission (seules neuf des 17 émoticônes sont affichées par défaut)

Si vous ne voulez pas d'émoticônes sur votre système, vous pouvez soit décocher la case "afficher dans le formulaire ?" (dans ce cas, elles n'apparaîtront pas dans les formulaires de soumission, mais seront toujours disponibles via leurs codes), soit utiliser le lien de suppression pour les retirer entièrement.
