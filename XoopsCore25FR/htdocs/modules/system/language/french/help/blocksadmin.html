Aide : Blocs

Description

Les blocs sont des sections discrètes de contenu qui peuvent être créées et configurées dans l'interface d'administration. Des blocs personnalisés peuvent être créés et sont généralement constitués de texte, de graphiques et d'images. Le contenu de ces blocs peut être formaté individuellement ou hériter du formatage du site. Les blocs peuvent aussi faire partie d'une sortie d'affichage de modules. La page d'administration des blocs est l'endroit où tous les blocs sont configurés et administrés.

Les blocs sont comme la première page d'un journal : ce sont des bande-annonces qui incitent les lecteurs à lire des histoires plus complètes dans les pages intérieures. Les blocs offrent de petits morceaux de contenu dynamique avec des liens vers le contenu réel stocké dans le module.

Une partie des choses qui se passent en arrière plan lorsqu'un module est installé dans un site XOOPS, c'est que les blocs du module sont enregistrés dans le système afin qu'ils puissent être utilisés à volonté par le ou la webmestre.

Que peut faire un ou une webmestre avec un bloc ? Eh bien, la plupart du temps, décider où le bloc sera affiché dans le site (l'équivalent de la définition d'une fenêtre de magasin, qui peut être à l'extérieur, en page d'accueil ou même à l'intérieur : un bloc peut être défini à être montré dans d'autres modules), et quand il est affiché, où dans une page, il sera vu.

Chaque module installé sur un site XOOPS enregistre ses blocs dans le système, et le ou la webmestre peut décider où il sera affiché, dans quelle partie de la page il apparaîtra et quel genre de pertinence, de poids ou de position il aura. L'administrateur ou administratrice du site peut également modifier l'étiquette du bloc et, si il ou elle est assez bien informé(e), peut même modifier le modèle qui sous-tend la présentation du bloc.

Tutoriel

Vous pouvez masquer le titre qui s'affichera lorsque le bloc sera affiché en ajoutant "//" (c'est à dire slash-slash-espace) comme les trois premiers caractères du titre. De cette façon, vous pouvez joindre plusieurs blocs de façon transparente pour construire la vue que vous souhaitez.

Vous pouvez créer vos propres blocs de contenu en utilisant le bouton "Add Block". Vous pouvez créer votre propre contenu en HTML, en texte brut avec des codes XOOPS, ou même en PHP. Dans vos blocs personnalisés, le code "{X_SITEURL}" sera remplacé par l'URL de votre site avec une barre oblique (slash).
