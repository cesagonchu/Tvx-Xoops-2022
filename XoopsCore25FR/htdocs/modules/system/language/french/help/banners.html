Aide : Bannières

Description

Les bannières sont des publicités affichées dans la partie supérieure de la page. Il peut s'agir d'images ou de films flash, comme par exemple la bannière XOOPS qui tourne en rond et qui fait partie de l'installation par défaut. XOOPS a un système intégré pour gérer les bannières et créer des comptes pour les clients publicitaires, qui se trouve dans le Module Système -> Bannières. Vous pouvez avoir plusieurs bannières sur un site XOOPS - la zone d'affichage tourne autour de chacune des bannières actives à chaque fois qu'une nouvelle page est chargée.

Chaque fois qu'une bannière est affichée pour un visiteur, cela compte pour une "impression", qui est la monnaie d'échange commune des bannières. Les clients paient généralement pour un certain nombre d'impressions, c'est-à-dire pour que leur publicité soit affichée un certain nombre de fois.

Remarque : si vous vendez des bannières en fonction du nombre d'impressions, il y a une chose importante que vous devez faire : Allez dans Module système -> Préférences -> Paramètres généraux et remplissez le champ "Votre adresse IP" (si vous en avez une). Cela empêchera vos visites sur le site d'être incluses dans le nombre d'impressions. Ne pas le faire pourrait être considéré comme quelque peu malhonnête (notez cependant que de nombreux administrateurs n'ont pas d'adresse IP permanente - par exemple ceux qui utilisent des modems téléphoniques).
