Aide : Modules

Description

Les modules sont des constructions de logiciels qui, en tirant parti de l'infrastructure fournie par XOOPS, permettent au webmestre d'offrir aux utilisateurs du site un contenu dynamique ou statique généré à l'aide d'une interface qui, le plus souvent, est très facile à utiliser.

Une partie de la puissance de XOOPS réside dans la façon dont les modules sont gérés. Le webmestre peut facilement installer, désinstaller, activer ou désactiver tout module spécifique (à l'exception du module Système, pour des raisons évidentes). Il est également possible de renommer les modules et de déterminer l'ordre dans lequel ils apparaîtront à l'utilisateur final dans le menu du site (si vous voulez qu'ils apparaissent tout court), car vous pouvez également fournir une fonctionnalité cachée, en définissant des modules actifs qui n'ont aucune présence visible pour l'utilisateur occasionnel. Par exemple, vous pouvez définir un module comme non visible dans le menu en désactivant l'icône de contrôle dans la colonne Menu. Ainsi, pour le module Nouvelles, vous pourriez envoyer un message privé à un groupe sélectionné d'utilisateurs et leur dire d'accéder à ce répertoire en tapant l'adresse dans le champ URL, quelque chose comme suit http://www.yoursite.com/modules/news.

Pour modifier l'ordre des modules (qui sera reflété dans le menu), il vous suffit de faire glisser et de déposer les modules à l'endroit souhaité.

L'installation des modules nécessite une certaine planification, à la fois pour les modules à afficher et pour la mise en forme des pages. C'est une bonne idée d'examiner d'autres sites XOOPS et de voir quels modules sont utilisés pour afficher différents types de contenu. Ceci est facilement réalisable en ouvrant n'importe quel site XOOPS et en survolant les liens de menu sur les panneaux latéraux gauche et droit. En même temps, regardez les adresses Internet affichées dans la barre d'état de votre navigateur. Vous devriez voir des adresses comme celle-ci : http://the_site.com/modules/news/.... Tous les modules sont stockés dans le répertoire /modules/. Ainsi, le secteur suivant de l'adresse (dans cet exemple "news") indique le module qui affichera le contenu de ce lien de menu.

Si vous ouvrez la page du lien, vous verrez un exemple de l'utilisation du module et du type de contenu qu'il affiche. L'examen d'autres sites de ce type vous donnera également un bon aperçu des possibilités de mise en page et de conception.
