Aide : Maintenance

Description

La maintenance fournit plusieurs fonctions de base pour vous aider à maintenir votre installation XOOPS et la garder en bonne santé :

- nettoyer le dossier cache
- vider la table de session
- effectuer la maintenance des tables de la base de données (analyser, vérifier, optimiser et réparer)

Vous pouvez également effectuer un "Dump" de la base de données entière ou des tables individuelles à des fins de sauvegarde.
